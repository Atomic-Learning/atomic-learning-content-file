The content.html file in an Atomic Learning page repository contains the main HTML content for the page. This file will be periodically read by a program on the Atomic Learning server and processed to update the page on the website.The contents of this file will be placed inside a larger template. This means it does not require the full setup of a HTML page, just the educational content.

Formatting Rules and Best-Practice

Paragraph Tags

Normal text on the page should be enclosed in paragraph tags, for example:

<p>This is a paragraph.</p>

<p>This is a second paragraph.</p>

Headings

Within the page, the <h1> heading is reserved for the title of the page, which is included as part of the template the content will be placed into, and is populated from the metadata.json file. It is encouraged to use <h2> to <h6> headings as appropriate to structure the content of the page, particularly for longer pages.

Styling

Styling of the content should be kept to a minimum as the Atomic Learning site uses global settings to style common elements such as tables in a consistent manner. As a result, the style tag should not be used in the content.html file.
